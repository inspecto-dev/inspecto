# React Rspack Legacy playground

This playground exercises the legacy Rspack integration using `@rspack/cli@0.3.x` together with `@inspecto-dev/plugin/legacy/rspack`.

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get started

Start the legacy Rspack dev server, and the app will be available at [http://localhost:3000](http://localhost:3000).

```bash
pnpm run dev
```

Build the app for production:

```bash
pnpm run build
```
